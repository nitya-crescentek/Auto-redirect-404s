Redirects all 404 errors to a custom URL or home page. Helps fix 404 errors in Google Search Console with proper SEO redirects.

== Description ==

Auto Redirect 404 to Custom URL is a simple plugin that automatically redirects all 404 (Not Found) errors to a custom URL or your site's home page. This helps improve user experience and SEO by preventing broken links and fixing 404 errors reported in Google Search Console.

**Features:**
* Redirect all 404 errors to a custom URL or home page.
* Easy to configure from the WordPress admin.
* Helps fix 404 errors in Google Search Console.
* SEO-friendly 301 redirects.
* Lightweight and fast.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/auto-redirect-404s` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'Settings' > 'Auto Redirect 404s' to configure the custom redirect URL.

== Frequently Asked Questions ==

= Can I redirect 404s to any URL? =
Yes, you can set any valid URL as the redirect target.

= Does this plugin use 301 redirects? =
Yes, the plugin uses SEO-friendly 301 redirects.

= Will this fix 404 errors in Google Search Console? =
Yes, redirecting 404s helps resolve errors reported in Google Search Console.
